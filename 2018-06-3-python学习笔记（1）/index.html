程序注释：

程序注释是编译器会忽略的文本，其主要作用是使阅读者便于理解程序。

  1. 多行注释

    '''………… '''or"""…………"""
多行注释是由3个单引号或三个双引号括起来的字符串
又称作文档字符串

  2. 单行注释

    #这是一行注释

    单行注释以#符号开头

数据类型

所有的值和对象都是有类型的，表达式中的运算数的类型会在运行时检查。

  1. 任何变量都可以指定任何类型的一个值
  2. 编写代码时不需要关心数据类型

导入模块

import语句使得另一个模块中的标识符可以被另一个程序所用。

  1. 导入一整个模块
import math #导入math模块

这使得在math模块中定义的任何名字，都可以在当前模块中通过math.这形式的语法而变得可用。

  1. 导入模块中的一个名称

    from math import sqrt #从math中导入sqrt函数

这样就可以直接使用sqrt而不需要使用math.sqrt这种形式

注意

学会使用help函数学习函数的使用方法

  1. 使用if name = “main“

    if _name_ = "_main_":

  1. 循环语句

    1. while

    2. for …… in ……：

      当运行这个循环的时候，他将可迭代的对象中包含的每一个值都赋值给循环变量，并且在每一个赋值上下文中运行语句的序列。
