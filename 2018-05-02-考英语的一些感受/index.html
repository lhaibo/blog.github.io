5-2 英语考完后的一些感触

发表于 2018-05-02 | 更新于 2019-05-25 | 分类于 blog

期中考试——英语

塞翁失马，焉知非福

时间过得飞快

不知不觉就到了期中

这一学期过了一大半了

就在今晚

打响了期中考试的第一场——英语

一直不太好的一门学科

不知道在我记了那么多单词，听了那么多听力后是否有进步

努力之后真的会有回报，也许会很慢，但只要坚持下去，在不经意的一天便会发现自己竟然进步这么多了

做任何事之前一定要做好准备

今天临近考试时突然意识到,许久没摸过的听英语听力的耳机是否出现问题，它还好不？
从抽屉里翻出它，按了按开机，竟然没反应。心想：难道是电池没电了？总不可能是耳机坏了吧。遂将电池盖打开，惊觉电池竟然泄漏了，电池里面的不明液体竟然都流出来了，不知道我的耳机坏了没，，，，

把室友耳机里的电池借过来一试，还好，额日记没坏，就电池坏了，去超市买两节电池就好。果断提醒其他室友也检查一下电池有没有坏。没想到，有两名室友的电池也泄漏了，由此观之，买耳机送的电池质量是真的差，以后一定要注意。

我本以为事情告一段落了

万万没想到，在考试途中，竟然也有电池坏的，导致整场考试的听力都没听，完全靠气运去蒙，不知他的气运如何，祝他好运吧！

耳机有风险，听力需谨慎

以后自己一定要多准备一些电池应急，大型考试还要考虑是否准备一个备用耳机，以防止因为各种意外导致一些很重要的事没有达到预期的目标

幸亏我是提前发现了耳机电池的问题，不然可能会和我同学一样，被老师说活该了，真是不幸中的万幸。

今天的两点收获：

1. 万事做足准备总是不会有坏处的
2. 塞翁失马，焉知非福，风水轮流转，永远不要嘲笑别人
