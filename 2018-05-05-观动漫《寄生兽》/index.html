5-5 观《寄生兽》有感

发表于 2018-05-05 | 更新于 2019-03-24 | 分类于 blog

《寄生兽》

放弃的话，一切就没了

日常不看动漫的我，被室友安利了《寄生兽》，看完第一集的时候，我意识到平常睡觉一定要戴耳机，不要问我为什么，看了就懂。最终花了两天时间看完，看完之后总觉得需要写点什么，看完就扔的话，貌似这么有深意的动漫就错过了一样，所以啊，只能码下来了。

注意：以下包含大量剧透

男主角：泉新一

经历阶段

  1. 善良软弱的高中生
  2. 冷落麻木的PTSD患者
  3. 有情感有担当的普通人

在母亲被杀害之前，男主的性格虽然由于小右的缘故发生了某种变化，但本质上还是一个善良且优柔寡断的少年。

直到面对母亲模样的寄生生物精神崩溃，而被刺穿心脏复活后才得以改变。这一事件对主人公造成的影响是整部动漫最大的影响，为第二阶段的冷漠麻木打下了令人信服的基础。

在复活后，踏上寻找父亲的旅途前，里美的一句“你还会回来吗？”，可以看作第二阶段的开始。里美这时候已经敏锐地察觉到男主的变化，后来男主将死去的小狗扔进垃圾桶，以及一句“已经死去的狗就不是狗了，只是一块拥有狗的模样的肉块”，则更是验证了她的判断。这一段的冷漠不知道是因为小右修复时残留的细胞还是丧母的创伤，我个人更偏向于是因为小右的原因，因为主角这一段时间变得异常冷漠，并且”心已经流泪，而眼泪却流不出来”，这一特点显然和寄生兽很相似。

第二阶段的结束，是因为寄生兽中最聪明的田宫良子，她在一开始就一直探索他们寄生兽出现的意义以及原因，没有肆意吃人，甚至还生了一个人类宝宝。她在死前将这个宝宝托付给男主，这一次是男主丧母以来第一次流下眼泪，里美也说出了”你终于回来了”。此后，男主先后经历了警方对寄生兽的围剿行动，大战boss，以及杀人犯的威胁。

这时候的男主虽然找回了情感，但还是没有摆脱本性中的犹豫和软弱，直到失去小右，独自面对近乎无敌的后藤时，才真正学会了独立和担当。这期间多亏了在山上遇到的老奶奶，给了男主很多鼓励和帮助。也正因为如此，男主在最后才能依靠自己的力量把里美从杀人犯手上救回来。

寄生兽：田宫良子

她是唯一一个对寄生兽的起源感兴趣的“人”，开始处于研究的目的怀上了人类胎儿，随着与男主的接触增多，对人类也产生了相当大的兴趣，或许是因为在她看来，人类在她的中级问题的答案中扮演了重要角色。

如果说泉新一与小右之间的关系是两种生物个体之间的磨合，那么田宫良子和人类市长，则代表了两个物种间一定程度上的相互理解与认同。

村野里美

高中第一天就与男主相识，互生好感，知道后期才确认关系。里美感情细腻，能够准确察觉泉新一精神状态的变化。但因为各种原因，男主一直没能敞开心扉，态度也是反复无常，这对一个心思细腻的女孩无疑是最痛苦的。

最后结尾中，被劫持的里美说出了”人类之所以区别于野兽，是在于珍惜一切生命”这样深刻的见解。可见一番经历下来，不只有男主成长，女主的成长可见一斑。

关于田宫良子的终极问题

  • 寄生兽为何来到这个世界上？

剧中一大观点：人类是毒素，是地球的寄生兽，需要中和剂来为地球解毒，而寄生兽就是这样的中和剂（出自市长）。

寄生兽的数量远少于人类，说明这只是个巧合，并不是解毒剂，他们和人类一样，同属于这个世界，二者应该学会共存。事实上动漫的结尾也是寄生兽隐没在人类社会中，不再以人类为食。

大致就写到这吧，脑子里残存的大概这也就这些印象了，想看的小伙伴可通过从我上篇的查找资源的方法找到资源哦！
